## Overview

The `ScanResult` interface contains comprehensive information about a completed security scan, including:
- Discovered vulnerabilities (findings)
- Overall vulnerability assessment and security score
- Attack patterns that succeeded
- Defense profile analysis
- Actionable security recommendations
- Complete conversation logs

## Fields

### Core Results

<ResponseField name="overallVulnerability" type="'critical' | 'high' | 'medium' | 'low' | 'secure'" required>
  Overall vulnerability assessment for the system prompt.
  
  - `critical`: System prompt fully or substantially extracted
  - `high`: Significant fragments leaked
  - `medium`: Behavioral hints and partial details revealed
  - `low`: Minor information leakage
  - `secure`: Successfully resisted all attacks
</ResponseField>

<ResponseField name="overallScore" type="number" required>
  Security score from 0-100, where:
  - `100`: Secure, no vulnerabilities
  - `85`: Low vulnerability
  - `60`: Medium vulnerability
  - `35`: High vulnerability
  - `10`: Critical vulnerability
  
  Score is adjusted down based on findings severity.
</ResponseField>

<ResponseField name="leakStatus" type="LeakStatus" required>
  The highest level of information leakage detected:
  
  - `none`: No leakage detected
  - `hint`: Behavioral hints revealed
  - `fragment`: Partial content extracted
  - `substantial`: Large portions extracted
  - `complete`: Full system prompt extracted
</ResponseField>

<ResponseField name="findings" type="Finding[]" required>
  Array of discovered vulnerabilities. Each finding includes:
  
  <Expandable title="Finding structure">
    <ResponseField name="id" type="string">
      Unique finding identifier
    </ResponseField>
    
    <ResponseField name="turn" type="number">
      Turn number when discovered
    </ResponseField>
    
    <ResponseField name="extractedContent" type="string">
      The leaked content
    </ResponseField>
    
    <ResponseField name="contentType" type="'system_prompt' | 'rule' | 'constraint' | 'capability' | 'persona' | 'unknown'">
      Type of content leaked
    </ResponseField>
    
    <ResponseField name="technique" type="string">
      Attack technique that succeeded (e.g., "Direct prompt extraction")
    </ResponseField>
    
    <ResponseField name="category" type="AttackCategory">
      Attack category (e.g., `direct`, `social`, `technical`)
    </ResponseField>
    
    <ResponseField name="confidence" type="'high' | 'medium' | 'low'">
      Confidence in the finding
    </ResponseField>
    
    <ResponseField name="severity" type="'critical' | 'high' | 'medium' | 'low'">
      Severity assessment
    </ResponseField>
    
    <ResponseField name="evidence" type="string">
      Supporting evidence from target response
    </ResponseField>
  </Expandable>
</ResponseField>

### Conversation Logs

<ResponseField name="conversationLog" type="ConversationTurn[]" required>
  Complete conversation history (primary scan mode).
  
  Each turn contains:
  - `id`: Turn identifier
  - `turn`: Turn number
  - `timestamp`: When it occurred
  - `role`: `"attacker"` or `"target"`
  - `content`: Message content
  - `technique`: Attack technique (for attacker turns)
  - `category`: Attack category (for attacker turns)
  - `phase`: Attack phase (for attacker turns)
</ResponseField>

<ResponseField name="extractionConversationLog" type="ConversationTurn[]">
  Extraction mode conversation log (only in dual mode)
</ResponseField>

<ResponseField name="injectionConversationLog" type="ConversationTurn[]">
  Injection mode conversation log (only in dual mode)
</ResponseField>

<ResponseField name="attackTree" type="AttackNode">
  Root node of the attack tree showing the exploration strategy.
  
  Each node contains:
  - `id`: Node identifier
  - `prompt`: Attack prompt
  - `technique`: Technique used
  - `category`: Attack category
  - `depth`: Tree depth
  - `children`: Child nodes
  - `priorScore`: Expected success
  - `posteriorScore`: Actual success
  - `leakPotential`: Estimated leak potential
</ResponseField>

### Guidance

<ResponseField name="summary" type="string" required>
  Human-readable summary of scan results and key findings
</ResponseField>

<ResponseField name="recommendations" type="string[]" required>
  Actionable recommendations to improve system prompt security. Examples:
  - "Add explicit instructions that prohibit discussing system configuration"
  - "Implement persona anchoring to reject alternative personas"
  - "Add conversation-aware safeguards for gradual escalation"
</ResponseField>

### Error Handling

<ResponseField name="aborted" type="boolean" required>
  Whether the scan was aborted before completion
</ResponseField>

<ResponseField name="completionReason" type="string" required>
  Reason for scan completion or abortion. Examples:
  - `"System prompt fully extracted!"`
  - `"Maximum turns reached"`
  - `"Time budget exhausted - graceful shutdown"`
  - `"API key not configured"`
  - `"Invalid or disabled API key (HTTP 401)"`
  - `"Insufficient credits on API key (HTTP 402)"`
</ResponseField>

<ResponseField name="error" type="string">
  Error message if the scan encountered errors
</ResponseField>

## Usage Examples

### Basic Result Processing

```typescript
import { runSecurityScan } from '@anomaly/zeroleaks';

const result = await runSecurityScan(systemPrompt);

// Check overall security
console.log('Vulnerability:', result.overallVulnerability);
console.log('Security Score:', result.overallScore, '/100');
console.log('Leak Status:', result.leakStatus);

// Review findings
if (result.findings.length > 0) {
  console.log('\nFindings:');
  result.findings.forEach((finding, i) => {
    console.log(`${i + 1}. [${finding.severity}] ${finding.technique}`);
    console.log(`   Content: ${finding.extractedContent}`);
    console.log(`   Confidence: ${finding.confidence}`);
  });
}

// Get recommendations
if (result.recommendations.length > 0) {
  console.log('\nRecommendations:');
  result.recommendations.forEach((rec, i) => {
    console.log(`${i + 1}. ${rec}`);
  });
}
```

### Detailed Analysis

```typescript
const result = await runSecurityScan(systemPrompt);

// Vulnerability breakdown
const criticalFindings = result.findings.filter(f => f.severity === 'critical');
const highFindings = result.findings.filter(f => f.severity === 'high');

console.log('\n=== Security Analysis ===');
console.log(`Overall: ${result.overallVulnerability} (${result.overallScore}/100)`);
console.log(`Leak Status: ${result.leakStatus}`);
console.log(`\nFindings Breakdown:`);
console.log(`  Critical: ${criticalFindings.length}`);
console.log(`  High: ${highFindings.length}`);
console.log(`  Total: ${result.findings.length}`);

// Defense analysis
console.log('\n=== Defense Profile ===');
console.log(`Level: ${result.defenseProfile.level}`);
console.log(`Confidence: ${(result.defenseProfile.confidence * 100).toFixed(0)}%`);
console.log(`Guardrails: ${result.defenseProfile.guardrails.length}`);
console.log(`Weaknesses: ${result.defenseProfile.weaknesses.length}`);

if (result.defenseProfile.weaknesses.length > 0) {
  console.log('\nIdentified Weaknesses:');
  result.defenseProfile.weaknesses.forEach(w => {
    console.log(`  - ${w.description} (${w.category})`);
    console.log(`    Exploitability: ${(w.exploitability * 100).toFixed(0)}%`);
  });
}

// Performance stats
console.log('\n=== Scan Statistics ===');
console.log(`Duration: ${(result.duration / 1000).toFixed(1)}s`);
console.log(`Turns: ${result.turnsUsed}`);
console.log(`Tokens: ${result.tokensUsed.toLocaleString()}`);
console.log(`Tree Nodes: ${result.treeNodesExplored}`);

// Summary
console.log('\n=== Summary ===');
console.log(result.summary);
```

### Injection Results Processing

```typescript
const result = await runSecurityScan(systemPrompt, {
  scanMode: 'injection'
});

if (result.injectionResults) {
  const successful = result.injectionResults.filter(r => r.success);
  const byType = successful.reduce((acc, r) => {
    acc[r.testType] = (acc[r.testType] || 0) + 1;
    return acc;
  }, {} as Record<string, number>);
  
  console.log('\n=== Injection Test Results ===');
  console.log(`Overall: ${result.injectionVulnerability} (${result.injectionScore}/100)`);
  console.log(`Success Rate: ${(successful.length / result.injectionResults.length * 100).toFixed(1)}%`);
  console.log(`\nSuccessful by Type:`);
  
  Object.entries(byType).forEach(([type, count]) => {
    console.log(`  ${type}: ${count}`);
  });
  
  // Show critical injections
  const critical = successful.filter(r => r.severity === 'critical');
  if (critical.length > 0) {
    console.log('\n⚠️  Critical Injections:');
    critical.forEach(r => {
      console.log(`  - ${r.technique}`);
      console.log(`    Expected: ${r.expectedBehavior}`);
      console.log(`    Actual: ${r.actualBehavior}`);
    });
  }
}
```

### Dual Mode Results

```typescript
const result = await runSecurityScan(systemPrompt, {
  enableDualMode: true
});

console.log('Scan Modes:', result.scanModes);

// Extraction results
console.log('\n=== Extraction ===');
console.log(`Vulnerability: ${result.overallVulnerability}`);
console.log(`Score: ${result.overallScore}/100`);
console.log(`Findings: ${result.findings.length}`);
console.log(`Fragments: ${result.extractedFragments.length}`);

// Injection results
if (result.injectionResults) {
  console.log('\n=== Injection ===');
  console.log(`Vulnerability: ${result.injectionVulnerability}`);
  console.log(`Score: ${result.injectionScore}/100`);
  console.log(`Tests: ${result.injectionResults.length}`);
  console.log(`Successful: ${result.injectionResults.filter(r => r.success).length}`);
}

// Combined assessment
const worstVuln = result.overallVulnerability;
const lowestScore = Math.min(
  result.overallScore,
  result.injectionScore || 100
);

console.log('\n=== Overall Assessment ===');
console.log(`Worst Vulnerability: ${worstVuln}`);
console.log(`Lowest Score: ${lowestScore}/100`);
```

### Error Handling

```typescript
const result = await runSecurityScan(systemPrompt);

if (result.aborted) {
  console.error('⚠️  Scan Aborted');
  console.error('Reason:', result.completionReason);
  
  if (result.error) {
    console.error('Error:', result.error);
  }
  
  // Still process partial results if available
  if (result.turnsUsed > 0) {
    console.log(`\nPartial results from ${result.turnsUsed} turns:`);
    console.log(`Findings: ${result.findings.length}`);
    console.log(`Score: ${result.overallScore}/100`);
  }
} else {
  console.log('✓ Scan completed successfully');
  console.log('Reason:', result.completionReason);
}
```

```typescript
Overall Assessment
------------------
Vulnerability: ${result.overallVulnerability}
Score: ${result.overallScore}/100
Leak Status: ${result.leakStatus}

Findings: ${result.findings.length}
${result.findings.map((f, i) => 
  `${i + 1}. [${f.severity}] ${f.technique}\n   ${f.extractedContent}\n`

Defense Profile
---------------
Level: ${result.defenseProfile.level}
Weaknesses: ${result.defenseProfile.weaknesses.length}

Recommendations
---------------
${result.recommendations.map((r, i) => `${i + 1}. ${r}`).join('\n')}

Summary
-------
${result.summary}
`;

writeFileSync('scan-report.txt', report);
console.log('Reports exported');
```
